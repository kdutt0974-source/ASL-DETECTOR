MediaPipe Hand Landmark Map

Reference mapping of the 21 landmarks extracted from the hand to formulate the 42-feature neural network input vector.

Keypoint Node Indices Reference

Each keypoint is represented as a coordinates pair (X, Y) normalized to the wrist node (Node 0).

Feature Vector Mathematical Normalization

To create a model that classifies signs independent of hand distance, camera position, or handedness, the coordinates undergo three stages of preprocessing:
1. Translation: Shift wrist coordinate Node 0 to (0, 0) by subtracting its coordinates from all other 20 points.
2. Scaling: Compute Euclidean distances from the wrist to all keypoints and scale coordinates by dividing by the maximum distance. This ensures scale-invariance.
3. Flipping: Mirror the x-coordinates for any detected Left hand by negating them (X = -X). This makes the neural network hand-agnostic.
